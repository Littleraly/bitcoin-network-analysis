# Bitcoin Network Activity Analysis

## Overview
This project analyzes Bitcoin blockchain data using Google BigQuery and Tableau to identify trends in network activity and transaction volume.

## Business Problem
How can we analyze Bitcoin network activity to identify trends and assess network growth over time?

## Key Insights
- Transaction activity shows steady growth over time  
- Transaction volume is highly volatile  
- Network usage remains consistent while economic value fluctuates  

## Tools Used
- SQL (Google BigQuery)  
- Tableau (Data Visualization)
